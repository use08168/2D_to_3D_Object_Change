# 05 · 배경차분 물체 추출 (무학습, 색 무관)

**참고:** arXiv:2012.07287 (Inpainting Error Maximization) — "배경은 예측 가능, 물체는 예측 불가 → 잔차가 곧 물체".

**목적:** 색/채도 임계 없이(흰·검·반투명 포함) 물체를 추출. 카메라 이동에도 강인.

## 방법
1. **빈 보드**를 정면(canonical)으로 편 기준을 만든다.
2. 매 프레임을 **현재 자세로 워프한 배경 예측**과 차분 → 잔차 = 물체. (원본 시점 유지: 정면 변환 안 함)

## 교훈
- **디지털 보드 렌더 차분은 실패**(마커 정렬오차로 잔차 과다) → 반드시 **실제 빈 보드 사진** 기준.
- 라이브에서 `r`로 같은 각도 빈 보드를 기준 잡으면 깨끗. `SHRINK`로 가장자리 잔차 억제.
- 정면(top-down) 변환은 세운 물체를 시차로 번지게 하므로 **이미지 공간 차분**이 유리.
