# 03 · 고전 필터 물체 분할 (무학습)

**목적:** ML/딥러닝 없이 색 필터 + 기하 정보로 보드 위 물체를 분할.

## 방법 (결합)
1. **물체 색 지정** — HSV 색창(ROI 샘플 자동 or 수동).
2. **보드-연결 필터** — 보드 영역에 *닿은* 최대 덩어리만 남김(키보드·손·배경 제거). 세운 물체는 보드 위로 솟으므로 자르지 말고 '연결'만 판정.

## 핵심 발견
- 이 웹캠에선 검은 보드칸도 어두운 파랑(H~107)이라 **색상(hue)만으론 물체와 안 갈림**. 물체(파란 립밤)는 **채도(S)가 훨씬 높아** S 임계(>120)가 진짜 판별자.
- 한계: 흰·검·무채색 물체는 채도로 못 잡음 → 05(배경차분)·07(깊이)로 해결.
