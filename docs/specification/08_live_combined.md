# 08 · 실시간 통합 (DA + FastSAM + ArUco)

**목적:** 세 모델을 강점에만 써서 실시간으로 물체 검출·측정. **정면 변환 없이 자연 웹캠 시점** 합성.

![composite](../images/08_camera_and_virtual_scene.png)
*왼쪽: 카메라 합성(검출+높이 히트맵). 오른쪽: 가상 3D 씬.*

## 역할 분담
| 역할 | 도구 |
|---|---|
| 감지 + 선/누움 판별 | DA V2 높이맵 |
| 완전한 실루엣 | FastSAM |
| 정확한 높이·길이·위치 | ArUco 기하 |

## 측정 정확도 (실측 대비)
| 물체 | 실측 | 측정 |
|---|---|---|
| 립밤 | 69 × ⌀19 | 69 · ⌀16 |
| 마커펜 | 78 × ⌀13 | 76 · ⌀13 |
| 볼펜(누움) | 148 × ⌀12 | 116 · ⌀11 |

높이/길이 오차 ±10% (DA-only −16~−45% 대비 크게 개선).

## 표시
- 원본(왜곡보정) 위에 물체 부위만 **높이 히트맵** 반투명 합성 + 윤곽/측정 라벨 + 우상단 높이맵 인셋.
- `run_live_combined`는 카메라 합성 | 가상 3D 씬을 나란히 표시.
